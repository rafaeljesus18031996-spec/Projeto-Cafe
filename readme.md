<h1>☕ Doce Aroma Café</h1>
<br>
<br>
<p>Bem-vindo ao Doce Aroma Café, um projeto de site desenvolvido para apresentar uma cafeteria moderna, aconchegante e apaixonada por café. O objetivo deste projeto é praticar habilidades em HTML e CSS, criando uma página responsiva e visualmente agradável.</p>
<br>
<h2>📖 Sobre o Projeto</h2>
<br>
<p>O Coffee House foi criado para simular o site de uma cafeteria, destacando seus produtos, ambiente e informações de contato.</p>
<br>
✨ Funcionalidades
<br>
<br>
Página inicial moderna e atrativa
Seção sobre a cafeteria
Apresentação dos cafés e produtos
Área de contato
Navegação simples e intuitiva
